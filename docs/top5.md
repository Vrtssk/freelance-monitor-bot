# Топ-5 актуальных объявлений

Кнопка/команда, которая показывает пользователю 5 самых релевантных объявлений
по его выбранным темам. Релевантность считается по формуле, учитывающей
4 фактора (см. `utils/relevance.py`).

## Точки входа (все 3 реализованы)

1. **Главное меню** — кнопка `🔥 Топ-5 актуальных` (`callback_data="menu:top5"`).
2. **Команда** `/top` — отправляет тот же список.
3. **Inline-кнопка под каждым уведомлением** — `🔥 Топ-5 актуальных`
   (`callback_data="menu:top5"`), ведёт к тому же списку.

Все три вызывают `bot/handlers/top.py` → `get_top_relevant` (выборка из БД по
пересечению тем) → `compute_relevance` (скоринг) → `format_top_list`.

## Формула релевантности

Итоговый score ∈ [0, 1], чем выше — тем актуальнее. Веса (`utils/relevance.py`):

| Фактор | Вес | Направление |
|--------|-----|-------------|
| Число откликов (`responses`) | **0.55** | меньше → лучше (0 откликов = максимум) |
| Сложность (`complexity`, 1–5) | 0.20 | проще → лучше |
| Свежесть (`recency`) | 0.18 | свежее → лучше |
| Цена (`price`) | 0.07 | минимальный, «на вырост» |

Ключевое правило из ТЗ: **0 откликов и сложность 3/5 важнее, чем 1/5 при
20 откликах** — поэтому отклики доминируют (вес 0.55). Проверено тестом
`test_zero_responses_complex_beats_many_responses_simple`.

### Откуда берутся данные

- `published_at` и `responses` — **парсятся из карточек бирж** (`scrapers/base.py`:
  `parse_relative_time`, `parse_responses`). Это best-effort эвристика по тексту
  карточки; если сайт меняет вёрстку, селекторы надо подстраивать под живой HTML.
- `complexity` — **эвристика по описанию** (`estimate_complexity`): простые
  ключевые слова (лендинг, вёрстка, скрипт) понижают оценку, сложные
  (микросервисы, БД, ML, интеграция, CRM) повышают; учитывается длина текста.
- `price` — `parse_price` из строки бюджета (₽/руб/$, «договорная» → None).
- Свежесть считается **динамически** в момент запроса: `age = now - published_at`,
  а если дата не спарсена — `now - seen_at` (когда пост впервые увидели).

Хранятся в `seen_posts` (`published_at`, `responses`, `complexity`, `price_value`);
скоринг происходит при каждом открытии Топ-5, поэтому свежесть всегда актуальна.
