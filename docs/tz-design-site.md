
Сайт должен выглядеть не как обычная админка, а как современный SaaS-сервис уровня Linear, Raycast, GitHub, Vercel или Notion. Минимализм, много воздуха, плавные анимации, красивые карточки, тёмная тема и ощущение "дорогого" продукта.

---

# Общая концепция

Стиль:

* Dark Modern UI
* Glassmorphism (очень умеренный)
* Неоновая подсветка только как акцент
* Скругления 16–20px
* Мягкие тени
* Градиенты
* Полностью адаптивный интерфейс

Ассоциации:

* Linear
* Vercel
* GitHub
* Arc Browser
* Raycast
* Discord (не по цветам, а по чистоте интерфейса)

Никаких Bootstrap.

Никаких стандартных таблиц.

Только красивые карточки и современный UI.

# Дополнительные "дорогие" детали

* Фоновый градиент с легким свечением в верхней части страницы.
* Едва заметная сетка (grid) или шумовая текстура на фоне, чтобы интерфейс не выглядел плоским.
* Стеклянные панели (glass effect) только для навигации и блока статистики, без перегрузки.
* Ленивая подгрузка карточек (infinite scroll или "Загрузить ещё").
* Автообновление новых объявлений без перезагрузки страницы с небольшой плашкой: «Появилось 3 новых объявления — Показать».
* Плавная анимация изменения статистики и фильтров.
* Закреплённая (sticky) верхняя панель при прокрутке.
* Переключатель светлой/тёмной темы (по умолчанию — тёмная).
* Сохранение выбранных фильтров в Local Storage.
* Возможность переключать вид между **Masonry** и **Compact Grid**.
* Отображение источника в виде фирменного цветного бейджа (FL.ru, Freelance.ru, Weblancer, Kwork), чтобы объявления легко различались визуально.
* Для каждой карточки — компактный блок с метаданными: дата публикации, время парсинга, бюджет, количество откликов, темы и оценка ИИ.
* Небольшой индикатор в шапке: **🟢 Мониторинг активен**, **⏱ Следующая проверка через 2:34**, **🔄 Последнее обновление 14:52:18**.
